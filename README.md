# Nexus AI

Elite AI Assistant with real phone control. Built with Expo/React Native + GPT-4o.

## Features
- 19 real device tools: camera, location, calls, Instagram, WhatsApp, YouTube, Twitter, Spotify
- Real AI streaming via OpenRouter/GPT-4o
- All iOS & Android permissions
